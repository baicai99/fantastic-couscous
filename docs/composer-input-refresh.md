# Composer 输入框改版说明（2026-03-07）

## 目标

- 参考截图对齐输入框形态：胶囊输入条、左侧 `+`、右侧发送主按钮。
- 保留既有发送、快捷选择器、动态提示词、高级变量等业务能力。
- 不引入语音能力，不调整页面整体背景主题。

## 交互映射

- 左侧上传入口：
  - 原“输入框上方上传行”迁移为输入条左侧 `+` 按钮。
  - 仍然触发同一隐藏文件输入控件，复用 `onSourceImagesAppend`。
- 中间输入区：
  - 继续使用 `Input.TextArea` 自动高度。
  - 普通 prompt 模式默认不展示占位文案；仅动态提示词模式展示简短模板示例占位（避免换行）。
  - 动态提示词 tag 在输入条上方轻微上移，减少遮挡输入区域。
  - 保留快捷触发与选择：
    - `/` 或 `、`：快捷功能项。
    - `@`：模型快捷补全。
    - `--`：参数命令补全。
  - Enter / Shift+Enter 行为保持不变。
- 右侧操作区：
  - 保留“高级变量”按钮与“发送”按钮。
  - 移动端将“高级变量”收敛为图标按钮，避免输入区挤压。

## 参考图区域

- 仍支持查看、单张删除、全部清空。
- 展示方式调整为输入条下方紧凑面板，减少对主输入条视觉干扰。
